# nest-auth-template

Простой шаблон nest-приложения со встроенной авторизацией.
Создан для работы с [react-auth-template](https://github.com/hackaton-templates/react-auth-template).

## Installation

1. Set environmental variables. See `.env.example`

2. Install packages
   ```bash
   $ npm install
   ```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
